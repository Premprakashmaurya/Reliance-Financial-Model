# Reliance Industries Financial Model

## Overview
This project presents a comprehensive financial model of Reliance Industries Ltd. built in Microsoft Excel.

## Objective
- Analyze the company's financial performance.
- Perform ratio analysis.
- Build a 3-statement financial model.
- Estimate intrinsic value using DCF Valuation.

## Features
- Income Statement
- Balance Sheet
- Cash Flow Statement
- Ratio Analysis
- Financial Dashboard
- DCF Valuation
- Forecasting

## Tools Used
- Microsoft Excel
- Financial Modeling
- DCF Valuation
- Ratio Analysis
